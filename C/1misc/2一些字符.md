>操作符
1. 算术操作符 +，-，*，/，%
2. 移位操作符：>>右移，<<左移。右移是存在两种，一种是逻辑右移（全部补0），一种是算数右移（补被舍弃的那个），可以用负数来测试，但是建议不要移动负数位，也不要写成<<-5，这种东西。
   ```
   int main()
   {
       int a =2;
       //向左移动一位，移动的是二进制位，a保持不变。
       //因为a是int,4B，即32b，所以int 2 = 00000000 00000000 00000000 00000010
       //左移，把整个向左移动1位，最后位补0变成
       //00000000 00000000 00000000 00000100 = 4
       int b =a<<1;
       return 0;
   }
   ```
3. 位操作符：&按位与， |按位或， **^按位异或（相同则0，相异则1），a^a=0,a^0=a**。这个也是在二进制下操作的，且必须只能在整数使用。
   ```
    int main()
    {
        //00000000 00000000 00000000 00000011
        int a =3;
        //00000000 00000000 00000000 00000101
        int b = 5;
        //00000000 00000000 00000000 00000001
        int c = a&b;

        //00000000 00000000 00000000 00000111 = 7
        int d = a|b

        //00000000 00000000 00000000 00000110 = 6
        int e = a^b;
        return 0;
    }
   ```
   >一道笔试题。交换两变量的值，但不允许使用第三个变量。

   ```
   int main()

   }
   ```

5. 单目操作符（即只对一个操作数操作的）：
   | name | meaning | name   | meaning                           |
   | ---- | ------- | ------ | --------------------------------- |
   | !    | 逻辑反  | sizeof | 操作符的类型长度(单位B)，不是函数 |
   | -    | 负值    | ~      | 对一个数的二进制位取反,如01变成10 |
   | +    | 正值    | --     |
   | &    | 取地址  | ++     |
   | ---  | ----    | *      | 解引用，间接访问操作符            |
   | !    | 逻辑反  | ()     | 强制转换                          |
   | -    | 负值    | &      | 取地址                            |
   | +    | 正值    |
   ```
   int main()
   {
       int a =0;
       //-1,一个整数的二进制表示由三种：原码，反码，补码。int在内存中存储的是补码。
       //对于负数-1来说：
       //10000000 00000000 00000000 00000001（源码，首位1表示符号位）
       //11111111 11111111 11111111 11111110（反码，保持符号位不变）
       //11111111 11111111 11111111 11111111(补码，源码取反+1
       //对于正数来说，三码一致
       //~0，变成：11111111 11111111 11111111 11111111这是补码，补码转换成源码则为-1
       cout<<~a;
       return 0;
   }
   ```

7. 逻辑操作符：&&,||
   >360面试题：请写出输出结果
   ```
   #include<stdio.h>
   int main()
   {
       int i =0,j=0,a=0,b=2,c=3,d=4;
       i = a++ && ++b && d++;
       //j = a++||++b||d++;
       printf("a = %d\n b=%d\n c=%d\n d==%d\n",a.b,c,d);
       return 0;
   }
   ```
   对于i：a++,不参与；++b参与；d++，不参与。i = 0&&3&&4 = 0；a = 1;b=3;c=3;d=5。

    **上述理解是错误的，因为当i=a++&&++b&&d++;的左边第一个为假时，当前语句的后面部分全不不执行，所以++b，d++根本没执行。如果调换位置把a++放后面，则前面语句便会执行。类似的，如果在判断||时，如果第一个语句就是真了，则他后面的语句全部不执行。**
